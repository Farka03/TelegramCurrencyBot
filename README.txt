# Телеграм бот для конвертации валюты
Это небольшой микросервис, созданный с помощью Spring Boot, который взаимодействует с Telegram Bot API и считает курсы валют (USD, KZT). Бот может конвертировать валюты на основе последних обменных курсов, полученных из открытого API.

## Функциональность
Бот предоставляет следующие основные возможности:
- Прием и отправка текстовых сообщений.
- Хранение всех полученных текстовых сообщений в базе данных PostgreSQL.
- Конвертация валюты на основе последних обменных курсов.

## Зависимости
В этом проекте используются следующие зависимости:
- [java-telegram-bot-api]: библиотека Java для взаимодействия с Telegram Bot API.
- [PostgreSQL]: выбранная база данных для хранения сообщений.

## Установка и использование
1. Github
1) Клонируйте репозиторий с Github.
2) Соберите приложение с помощью maven.
3) Настройке базу данных PostgreSQL в application.properties.
4) Получите API токен от Telegram Bot API.
5) Обновите токен бота Telegram.
6) Запустите приложение при помощи Maven.
7) После запуска можно найти бота в Telegram и начать его использовать. Для этого в боте пишем, к примеру - 1 USD, чтобы конвертировать в тенге или 500 KZT, чтобы конвертировать в доллары.

2. Docker
1) docker build ...
2) docker-compose up
